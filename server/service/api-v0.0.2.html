/User

/User/register post

注册

post

post /User/register

  • Request
  • Response

Query Parameters

  • phoneNumber: required (string)

    必须为正确的手机号格式

    Example:

    15851813132
  • password: required (string)

    限定最小长度

    Example:

    abc123
  • type: required (integer)

    0表示货主, 1表示司机

HTTP status code 200

Body

Type: application/json

Example:

{
  "result":0(失败)1(成功),
  "errorMsg":(表示失败的原因，比如手机号已经注册)
}

/Shipper

/Shipper/placeOrder post

下单

post

post /Shipper/placeOrder

  • Request
  • Response

Query Parameters

  • token: required (string)

    用于识别用户

    Example:

    sdfs234rrjkn2IFO
  • addressFrom: required (string)

    发货地

    Example:

    汉口路22号
  • addressTo: required (string)

    收货地

    Example:

    汉口路55号
  • loadTime: required (string)

    装车时

    Example:

    2015-11-09 00:00:00,注意格式
  • goodsType: required (string)

    货物类型

    Example:

    食品
  • goodsWeight: required (integer)

    货物的重量,单位是千克

    Example:

    4
  • goodsSize: required (integer)

    货物的体积, 单位是方

  • truckType: required (integer)

    所需车辆类

    Example:

    1
  • truckCapacity: required (integer)

    所需车辆载重类型

    Example:

    3
  • payType: required (integer)

    支付方式

    Example:

    0(线上)/1(线下)
  • price: required (integer)

    酬劳

    Example:

    100

HTTP status code 200

Body

Type: application/json

Example:

{
  "result":0(失败)1(成功),
  "errorMsg":失败原因
}
