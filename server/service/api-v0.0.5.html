/user/register post

注册

post

post /user/register

  • Request
  • Response

Query Parameters

  • phoneNumber: required (string)

    必须为正确的手机号格式

    Example:

    15851813132
  • password: required (string)

    限定最小长度,大于等于8

    Example:

    abc123
  • type: required (integer)

    0表示货主, 1表示司机

HTTP status code 200

Body

Type: application/json

Example:

{
  "result":0(失败)1(成功),
  "errorMsg":(表示失败的原因，比如手机号已经注册)
}
